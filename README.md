# Sakila SQL Queries - Final Assessment (Module 2)
## Description
This repository contains the solution to the Module 2 (Web and Database Data Extraction) final assessment.

The goal of this assessment is to evaluate my SQL queries knowledge using the Sakila database, through practicing with exercises including the following SQL concepts such as:

- Filtering data (WHERE, BETWEEN, LIKE)
- Aggregations (COUNT, AVG, GROUP BY)
- Joins (INNER JOIN, LEFT JOIN)
- Subqueries 
- Common Table Expressions (CTE)
- Set operations (UNION)

All queries are implemented in the file: Mod2_Final_Assessment.sql


## Technologies Used
- MySQL
- Sakila Database
- SQL


## How to Run the Project
1. Install MySQL
2. Load the Sakila database
3. Open the file: Mod2_Final_Assessment.sql
4. Run the queries in your SQL client (MySQL Workbench or similar)
